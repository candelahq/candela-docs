# Candela Desktop

Candela Desktop is a Flutter-based desktop application for managing LLM providers and visualizing traces.

## Features

- **Provider Management** — Add, configure, and switch between LLM providers
- **Trace Viewer** — Visualize distributed traces with span details
- **Cost Monitoring** — Track token usage and estimated costs
- **Secure Storage** — API keys stored in OS keychain
